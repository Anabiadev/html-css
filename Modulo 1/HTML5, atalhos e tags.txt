para colocar um número elevado como x7,(texto sobre escrito) use a tag <sup>

-------------------------------------------------------------------------------------------
                                          PARA LINKS

para abrir uma nova janela quando alguém clicar no seu link, utilize a tag target=" _blank"
rel="external">

use control + espaço para tags como <a>, dentro de href"".
use a tag <target _self> para quando alguém clicar no link, abrir na mesma página.
para links internos, não precisa colocar a URL inteira, porém isso só funciona se o link
estiver na mesma pasta.
para seguir para próxima página, use o rel= "next"
para voltar para página anterior, use o rel= "prev"
para usar um link que o mecanismo de busca não associe ao seu site, como links patrocinados, utilize rel= "nofollow"

caso queria adicionar um documento HTML com a tag <a>, mas esse documento não está na pasta, use:
../ para voltar para pasta anterior
./ é a própria pasta

para usar um documento em outra pasta, copie esse documento para a pasta que você quer.
------------------------------------------------------------------------------------------- 
                                        PARA IMAGENS

quer colocar um texto do lado de uma imagem? faça isso:
<img,<p> </p>

use a tag <picture> com <img>
para fazer com que uma imagem se expanda use a tag <source media max width 700px "imagens/foto-p.png" type.

nao se esqueça: siga a ordem das imagens colocando os <source>, (por exemplo, uma imagem de tamanho 700 fica em primeiro e a de 1000 fica em segundo, se inverter isso o código nao funciona.)
-------------------------------------------------------------------------------------------
                                          PARA AUDIOS

para usar um áudio no site que seja gratuito, use o youtube studio e vá em biblioteca
de audio.

use a tag <audio> e source + type para colocar um áudio em seu site.
alguns navegadores não suportam o formato mp3, então tem outras opções como WAV, OGG
para adicionar esses formatos, faça isso:
<source src="midia/guanacast-33.mp3" type=audio/mpeg">
para adicionar outros formatos mude o mp3: 
type="audio/ogg" 
type="audio/wav" não use wav porque ele é pesado!

a ordem é importante!
não use "None" ou "auto" no seu site, use metadata.
para audios curtos e se quiser repetir eles, use <loop>

-------------------------------------------------------------------------------------------
                                           PARA VIDEO
dicas para diminuir videos:
acesse o site handbrake e baixe o app

dica: use todos os formatos de videos no seu site: ogv, m4v e mp4.

para inserir um video no site, use a tag <video> , source src e type.
para diminuir a largura use "width"
não esqueça da tag "controls"
para colocar uma capa no seu video, use a tag <poster>

Pode gerar um custo muito grande para seu cliente se vc usar o código de uma forma
em que ele carregue o video em vários formatos como ogv,
por ser muito pesado. por isso, use a técnica de adicionar
videos de serviços de hospedagem de videos! (como Youtube)

depois de pegar o video no Youtube, clique em compartilhar > incorporar.
depois, é so dar control + v no VScode.

outra opção é o Vimeo, ele é pago, sua vantagem é sua ótima qualidade e vc pode escolher
entre videos que sao privados ou publicos, mas sua desvantagem é que o algoritmo dele
nao é tao bom quanto do Youtube.

se quiser colocar um video do Vimeo no seu código, use: 
vá para o site do Vimeo, clique no video > compartilhar > incorporar.
depois, é so dar control + v no VScode.
